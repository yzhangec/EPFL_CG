\subsection{Detailed Description}
This class loads and raytraces scenes consisting of cameras, lights, and objects

This class implements a sphere object, which is specified by its center and its radius. This class overrides the intersection method \hyperlink{structObject_ac6549d30793a9b3da070234bf10c7e91}{Object\+::intersect()}.
